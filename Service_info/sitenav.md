# Навигация по админ-панели (sitenav.md)

Полное дерево пунктов меню с описанием функционала.

### Основное меню

1.  **Рабочий стол (Dashboard)**
    - Общая статистика по заказам, выручке и курьерам.

2.  **Заказы (Orders)**
    - **Все заказы**: Список всех заказов из iiko.
    - **Детали заказа**: Просмотр состава заказа, адреса, статуса.

3.  **Клиенты (Customers)**
    - **Все клиенты**: Список всех гостей.
      - **[NEW] Панель обслуживания**: Кнопки "Проверить ID", "Слияние по UID", "Слияние по телефону".
      - **[NEW] Фильтры дубликатов**: Кнопки для поиска клиентов с одинаковыми телефонами или UID.
    - **Карточка клиента (Премиальная 8-вкладочная)**:
      - **1. Анкета**: Основные данные (ФИО, пол, ДР, регистрация), маркетинговые статусы и категории iiko.
      - **2. Бонусы**: Текущий баланс (iiko Card), кнопки начисления/списания баллов, полная история транзакций.
      - **3. Заказы**: Глубокая история за 12 месяцев из iiko Cloud. Возможность просмотра деталей заказа iiko или локальной карточки заказа.
      - **4. Аналитика**: [NEW] Глубокая статистика из iiko Server (OLAP): LTV, средний чек, частота, жизненный цикл (первый/последний заказ). Графики посещений.
      - **5. Риски**: Управление статусом "Высокий риск", указание причины. Визуальные индикаторы в шапке (Green/Yellow/Red).
      - **6. Адреса**: История адресов доставки с возможностью переключения статуса активности (Актуальный/Неактуальный).
      - **7. Уведомления**: Настройка маркетинговых подписок (Promo, Loyalty, Order Status) и логи отправленных сообщений.
      - **8. Дополнительно**: Техническая информация, RAW JSON данные из iiko для отладки.
    - **Синхронизация**: Кнопка массовой синхронизации и кнопка обновления в карточке.

4.  **Система (System)**
    - **Метки**: Управление тегами клиентов.
    - **Настройки iiko**: Ключи API и ID организации.
    - **Настройки Ботов**: Настройка Telegram и VK.
    - **Фоновые процессы**: [NEW] Мониторинг задач Celery, пауза, отмена и управление временными файлами импорта.
    - **Логи синхронизации**: Логи пачек синхронизации гостей.
    - **Логи системы**: Технические логи ошибок.
    - **Обслуживание базы (Maintenance)**: [NEW] API-инструменты для поиска клиентов без ID и объединения дублей по UID/телефону.

5.  **Отчеты (Reports)**
    - **Отчеты курьеров**: Детализация по доставкам.
    - **Выручка (OLAP)**: Финансовые отчеты.

6.  **Администрирование**
    - **Сотрудники**: Данные из iiko.
    - **Пользователи**: Доступ в админ-панель.

### Логика работы

- **Рекурсивная синхронизация**: Обрабатывает гостей пачками по 10 человек. Пауза 30 секунд. При возникновении ошибки процесс полностью останавливается до ручного вмешательства.
- **Ленивая синхронизация (On-Demand)**: Данные гостя актуализируются из iiko при:
  - Открытии карточки клиента в админ-панели.
  - Поступлении нового заказа от клиента.
  - Изменении бонусного баланса (транзакции).
  - [NEW] Ручном запуске синхронизации.
- **Управление задачами**: На странице "Фоновые процессы" можно контролировать выполнение массовых операций.
