
## Таблица courier_orders
| Поле | Тип | Описание | Пояснения |
|------|----------|----------|-----------|
| iiko_id | String | Уникальный ID записи | Формируется как `olap_{order_num}_{date}` для исключения дублей при повторной синхронизации. |
| employee_id | Integer | Ссылка на сотрудников | Связывает заказ с конкретным курьером из таблицы `employees`. |
